  • 调用地址：http://wechat.palm-h.com/h5/s6/cms/index.php/Home/index/s6_data
  • 请求方式：
  • 参数：
    • username:姓名
    • sex:性别
    • mobile:手机号
    • verifyMsg:验证码
    • city:城市
    • area:地区
    • address:详细地址

{

    "info": "提交成功",
    "uid":抽奖标识（存到cookie，抽过后就不能中奖了）(数据库id号码)
}
